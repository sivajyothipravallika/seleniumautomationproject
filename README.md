# seleniumautomationassignment
This is a selenium-java based assignment

## Project Description
This assignment is a UI automation framework built with Selenium WebDriver, Java, Maven, and TestNG to validate the login journey of the Experian application. The project automates browser-based login scenarios and is structured to support both local execution and cloud execution through LambdaTest.

Technically, the solution has been implemented using the Page Object Model to separate test flow from page interactions and improve maintainability. Test data is driven from multiple external sources including JSON, CSV, and Excel files. The framework also includes configurable browser and environment handling, headless execution support, TestNG suite execution, retry and listener support, Log4j2 logging, screenshot capture on failure, and ExtentReports-based reporting.

## Prerequisites
- Java 11+ (Selenium 4.x is not compatible with Java 8)
